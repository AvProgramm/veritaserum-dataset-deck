Collection discipline

The dataset starts with permission-not scraping

Before collection

  • Review robots.txt and site terms
  • Confirm copyright and redistribution rights
  • Record Monash ethics and privacy decisions
  • Seek written permission where terms are restrictive

Leak-resistant annotation rules

  • Use fact-check pages for claim wording, published ratings and discovery leads-not model evidence.
  • Prefer evidence that is independent and published before the claim.
  • Retain each portal’s original rating and document every mapping to Supported / Refuted / NEI.
  • If full text cannot be redistributed, publish identifiers, URLs, lawful excerpts and annotations.
  • Preserve dates, source authority, versions and reviewer decisions for every record.

Terms reviewed: MyCheck page terms · AFP Malaysia legal notice · AVeriTeC temporal-evidence principle

System readiness

Malay accuracy is plausible-but not yet demonstrated

The repository contains two verification paths with very different language readiness.

Pilot candidate

/factcheck/document

  • Whole-document Qwen3 classification
  • Tavily web evidence retrieval
  • No hard-coded language field
  • But benchmarks and query scaffolding are English
Needs adaptation

/verify sentence path

  • English spaCy sentence segmentation
  • English-oriented claim detector
  • Not suitable as the Malay annotation baseline
  • Requires Malay-specific validation or replacement
Use the document endpoint for the pilot, and report measured performance-not assumed multilingual capability.

Recommendation

Proceed with Malay-conditionally

The evidence-grounded gap is real, the sources are promising, and the existing document pipeline can support a pilot. Scaling depends on two gates.

Gate 1

Source permission

Gate 2

Two Malay reviewers

If both pass

Build the 50-claim pilot

Deliver before the next discussion

  • Source feasibility ledger: permission, terms, robots, structure and usable claim count
  • Internal 50-claim pilot with auditable provenance and no fact-check-page evidence leakage
  • Malay system report: macro-F1, evidence coverage, kappa, failures, latency and representative errors
  • One-page annotation guide and a go / revise / pivot decision
