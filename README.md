# Veritaserum — Dataset Literature Review Deck

An HTML presentation on fact-checking / claim-verification datasets: the
current landscape, gaps that motivate a new dataset, easy-to-collect English
domains, and an assessment of underrepresented Southeast Asian languages
(recommending Malay).

**View it live:** https://avprogramm.github.io/veritaserum-dataset-deck/

Team MCS13 · FIT3161/3162 Final Year Project · Monash University.
Navigate with arrow keys / space; press **N** for speaker notes.
